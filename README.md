# Online Book Store

This is a React-based web application for an online bookstore where users can browse, add books to a shopping cart, and authenticate using a basic login form.

## Features
- Book listing page with details
- Add books to cart functionality
- Simple login form for user authentication
- Responsive design

## Tech Stack
- **Frontend**: React, JavaScript, HTML, CSS
- **Version Control**: Git & GitHub

## Available Scripts
In the project directory, you can run:

### `npm start`
Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

### `npm run build`
Builds the app for production to the `build` folder.
